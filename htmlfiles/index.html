About

This Website is just for demostrating that the server can handle http requests and can send different kinds of files to its client.


You can try clicking on the tabs on the left to access few test files i added to the server to check if the server works fine


There is a tab named "Invalid URL". This is an error page which will be shown to the client if the requested file is not hosted by the server or, its location is changed. Though it is not perfect and can show you this page otherwise also.
